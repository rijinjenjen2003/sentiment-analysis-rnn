# Sentiment Analysis using RNN

## 📌 Overview
This project builds a sentiment analysis model using Recurrent Neural Networks (RNN) to classify text as positive or negative.

## 🚀 Features
- Text preprocessing (padding, encoding)
- Embedding layer for word representation
- RNN model for sequence learning
- Binary classification (positive/negative)

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- NumPy

## 📊 Dataset
- IMDb Movie Reviews dataset

## 🎯 Results
- Achieved ~80% accuracy on test data

```bash
pip install tensorflow numpy
python model.py
